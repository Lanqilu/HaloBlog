|  Gateway    |  网关 |

|  Message    |  消息管理服务 | 
|  Web        |  提供前台API接口服务 | 
|  Monitor    |  系统监控 | 
|  Base       |  基类 | 
|  Search     |  搜索服务 | 
|  Picture    |  图片服务 |
